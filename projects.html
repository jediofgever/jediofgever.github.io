Autonmous Lawn Mover

This robot is designed to mow grass autonmously. The robot has 2 cameras, GPS, Gyros, Acceleoreometrs, Raspberry PÄ° 3, 12W solar panel Arduino Mega 2560 , 3 Dc motors for locomotion and cutter blade.

With the two frontal webcams, robot is able to detect grass area, and unmowed grasses. Also this two webcams used for an obstacle avoidance algorithm. The upper camera detects area with the grass and obstacles that are needed to be avoid.

With GPS the robot avoids replication of same positions. Thus the operation of mowing grass, is done with optimal efficiency.

Computer vision algorithms are developed in Python and OpenCV and Embedded to Raspberry Pi. Since the computation power of Raspberry Pi is relatively limited the algorithms were not performed optimally.

At bottom left picture, the computer vision algorithm has deteceted intersted area , also the obstactle front of robot. The red line is meant to be optimal steering angle for the navigation. At bottom right picture GPS data is processed to an image, processed areas are circled in red line this indicated that the robot shall not pass through this area. This is done by comparing the most recent data from GPS with the scanned area.
Python code for computer vision is: lawnMowerCV.py
Arduino code for sensors, dc motors,PID is: lawnMower.ino
